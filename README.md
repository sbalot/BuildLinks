## About The Project

[![Product Name Screen Shot][product-screenshot]](https://example.com)

The BuildLinks ontology contains terminologies for defining the relationships of links identified by any matching approach. It consists of all necessary classes and properties for describing commonly required metadata for links. Existing domain ontologies for defining the provenance information and link information such as [prov](https://www.w3.org/TR/prov-o/), xx were also used.

The documentation for the ontology can be found at https://sbalot.github.io/BuildLinks/.


This project was developed as a part of research persistently storing link relationships identified between different heterogeneous resources used in the AEC domain. These resources include partial domain models, images and documents.
